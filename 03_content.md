# Content

Content fills structure — but does not define it.

In CAAF IF, content is treated as **bounded information**, not as free text.

## Content rules

Content should:
- serve the defined intent
- fit the given structure
- avoid redundancy across sections

Content should not:
- introduce new intent
- compensate for unclear structure
- rely on external context to be understood

## Content density

More content does not equal more clarity.
CAAF IF favors **precision over volume**.
