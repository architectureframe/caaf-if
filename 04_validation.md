# Validation

Validation ensures that intent, structure, and content remain aligned.

CAAF IF provides **evaluation criteria**, not scoring systems.

## Core validation questions

- Is the intent explicit and explainable?
- Does the structure clearly reflect the intent?
- Is each content element necessary?
- Can the asset be understood without external explanation?

## Validation outcome

Validation does not produce:
- rankings
- certifications
- performance metrics

It produces **structural confidence**.
